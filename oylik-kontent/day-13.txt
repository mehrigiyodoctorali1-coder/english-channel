⏰ 09:00 — POST 1
🔧 Conditionals: 1, 2, 3-tip

• Zero — har doim haqiqat: If you heat ice, it melts.
• First (1) — real kelajak: If it rains, I will stay home.
• Second (2) — xayoliy hozir/kelajak: If I were rich, I would travel.
• Third (3) — o'tmishdagi afsus: If I had studied, I would have passed.

⏰ 12:00 — POST 2
💬 Struktura (formula)

• 1-tip: If + Present Simple, ... will + verb
• 2-tip: If + Past Simple, ... would + verb
• 3-tip: If + Past Perfect, ... would have + V3

Eslatma: 2-tipda "I were" ishlatiladi ("I was" emas): If I were you...

⏰ 15:00 — POST 3
🎯 Qaysi tip?

• Real, ehtimoliy → 1-tip: "If I have time, I'll call you."
• Xayoliy, ehtimoldan uzoq → 2-tip: "If I won the lottery, I'd buy a house."
• O'tmish, o'zgartirib bo'lmaydi → 3-tip: "If she had left earlier, she wouldn't have missed the train."

Speaking'da 2 va 3-tip yuqori grammatik darajani ko'rsatadi.

📝 Homework + Useful word

To'g'ri shaklni qo'ying:
1. If I ___ (be) you, I would apologise.
2. If it ___ (rain) tomorrow, we will cancel.
3. If they had known, they ___ (come).
(Javob: 1 were, 2 rains, 3 would have come)
